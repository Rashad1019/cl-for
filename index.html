CAPABILITIES

MOVE FAST

⚡
ZERO CONFIG

Drop a forge.js in your project root. Auto-detects tasks, entry points, and dependencies without a single config line.

CORE
⚙
PARALLEL TASKS

Run dozens of tasks concurrently. Forge builds a dependency graph automatically and saturates your CPU cores.

PERF
◈
SMART CACHE

Content-addressed caching at the file level. Never rebuild what hasn't changed. Local and remote cache out of the box.

PERF
↻
WATCH MODE

Debounced file watcher with dependency tracking. Change a file, only related tasks re-run. Sub-100ms feedback loops.

WORKFLOW

SIMPLE
BY DESIGN

01
INSTALL GLOBALLY

One command. No peer dependencies, no lockfile conflicts, no post-install scripts.

02
DEFINE TASKS AS FUNCTIONS

Write a forge.js with named export functions. Each function is a task — no API to memorize.

03
RUN ANYTHING, ANYWHERE

Call forge <task> from any subdirectory. Forge walks up the tree to find your forge.js.

04
SHIP WITH CONFIDENCE

CI mode gives structured output, non-zero exits on failure, and automatic cache priming.
